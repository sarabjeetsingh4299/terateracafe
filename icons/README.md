# Icons Folder

This folder should contain the PWA icons for the Tera Tera Café app.

## Required Icon Sizes

Please add the following icon files for full PWA support:

- `icon-72x72.png` (72x72 pixels)
- `icon-96x96.png` (96x96 pixels)
- `icon-128x128.png` (128x128 pixels)
- `icon-144x144.png` (144x144 pixels)
- `icon-152x152.png` (152x152 pixels)
- `icon-192x192.png` (192x192 pixels)
- `icon-384x384.png` (384x384 pixels)
- `icon-512x512.png` (512x512 pixels)

## Icon Design Guidelines

- Use your café logo or a coffee-themed icon
- Ensure the icon works well on both light and dark backgrounds
- Make it simple and recognizable at small sizes
- Use PNG format with transparency if needed
- Consider the "maskable" icon requirements for Android

## Temporary Solution

Until you add actual icons, the website will work fine but may show default browser icons for PWA installation.

You can create icons using:
- Canva
- Adobe Illustrator
- Online icon generators
- PWA icon generators
